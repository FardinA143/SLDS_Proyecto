# SLDS_Proyecto

Proyecto para SLDS sobre software libre para móviles

## Descripción

Este proyecto explora el tema del software libre en dispositivos móviles, con un enfoque especial en sistemas operativos libres, aplicaciones y su impacto en la privacidad, seguridad y sostenibilidad. Incluye un documento principal en LaTeX que analiza alternativas libres a sistemas propietarios como Android e iOS, así como una presentación Beamer para exposiciones.

## Contenido del Proyecto

- **documento.tex**: Documento principal en formato LaTeX (report) que cubre introducción, sistemas operativos libres, aplicaciones de software libre, modelos de negocio y seguridad.

- **presentacion.tex**: Otra presentación Beamer, posiblemente una versión alternativa o complementaria.


## Cómo Compilar

### Documento Principal
Para compilar el documento `documento.tex` a PDF:
```bash
pdflatex documento.tex
```

### Presentación
Para compilar la presentación `presentacion.tex` a PDF:
```bash

pdflatex presentacion.tex
```

Asegúrate de tener instalado LaTeX (por ejemplo, TeX Live o MiKTeX) y los paquetes necesarios como babel para español, graphicx, hyperref, etc.
